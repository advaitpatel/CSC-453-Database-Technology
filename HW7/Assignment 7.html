Assignment 7

Recreate part of Assignment 5 using Java (JDBC):

  • Only implement the vendor portion (including the insert), using vendors7 as the table name.

  • Omit all code relating to invoices, and the ALTER statements for the vendor table.

  • No need to use PL/SQL as you can use Java variables and constructs.

  • List at least two additional fields (other than the ID and name) from the vendor table.

  • List all vendors, not just the added one. Order the listing by the vendor id descending.

  • The output should be in a report format, with column headings, aligned data, and reasonably limited line width.

  • Get the column headings from the metadata. Do not hard-code them in the program.
    You can hard-code the column widths to reduce the size if needed, but at least one width before the last field must be obtained from the metadata.
    You can change the column headings with an alias in the SQL.

  • Code a finally block which deletes the added table and closes the connection.

  • Do not hard-code your user-name or password in the program. Do not prompt for them. Use parameters.

Note that statement.executeUpdate() is required for the SQL DDL statements.
